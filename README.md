# 🚀 ASP.NET MVC Web Application

## 📌 Overview

This project is a web application developed using the ASP.NET MVC framework, modular design patterns.

It demonstrates a scalable backend structure, maintainable code organization.

The main goal of this project is to build a solid foundation for enterprise-level web applications while ensuring readability, flexibility, and performance.

---

## 🏗️ Architecture

The project is structured based on the **Model-View-Controller (MVC)** design pattern:

* **Models:** Handle data structure and business logic
* **Views:** Responsible for UI rendering
* **Controllers:** Manage application flow and user interactions

This separation ensures:

* High maintainability
* Better testability
* Clear code organization

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ozgurdh/Hospital_Management_Website.git
```

### 2. Open the project

Open the solution file (`.sln`) using Visual Studio.

### 3. Restore dependencies

```
Right click on solution → Restore NuGet Packages
```

### 4. Run the project

Press `F5` or click **Run** in Visual Studio.
